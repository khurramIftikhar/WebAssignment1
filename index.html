Introduction

This website is part of our BSCS group assignment and represents a simple multi-page HTML project created without using CSS. It contains different pages including hobbies, media, and contact information. Each page demonstrates the use of semantic HTML elements, navigation links, tables, lists, and embedded content. The goal is to practice structuring webpages using only HTML.
